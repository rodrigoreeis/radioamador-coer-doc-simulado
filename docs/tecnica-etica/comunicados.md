# Anexo 1, Seção 6 – Comunicados

## 6.1 Padronização: por que é necessária

O radioamadorismo é um serviço internacional. Dois radioamadores de países diferentes, falando línguas diferentes, precisam conseguir se comunicar. Para isso, existem padrões internacionais adotados por todos:

- **Alfabeto fonético internacional**: para soletrar indicativos e palavras
- **Código Q**: abreviações de três letras para comunicações rápidas, especialmente em CW
- **Procedimentos operacionais padronizados**: chamada, resposta, encerramento

## 6.3 Código Q – Principais Códigos

O Código Q é um sistema de abreviações de três letras (todas começando com Q) que permitem comunicações rápidas e eficientes, mesmo entre operadores de idiomas diferentes. Foi desenvolvido para a telegrafia marítima e é universalmente adotado no radioamadorismo.

| Código | Significado (pergunta) | Significado (resposta/afirmação) |
|--------|----------------------|----------------------------------|
| **QRA** | Qual é o nome de sua estação? | O nome de minha estação é _____ |
| **QRG** | Qual é minha frequência exata? | Sua frequência exata é _____ kHz/MHz |
| **QTH** | Qual é sua localização? | Minha localização é _____ |
| **QSL** | Pode confirmar o recebimento? | Confirmo o recebimento |
| **QRO** | Devo aumentar a potência? | Aumente a potência |
| **QRP** | Devo diminuir a potência? | Diminua a potência |
| **QTC** | Tem mensagens para transmitir? | Tenho _____ mensagens para _____  |
| **QRM** | Está sendo perturbado por interferência? | Estou sendo perturbado por interferência |
| **QRN** | Está sendo perturbado por estáticas? | Estou sendo perturbado por estáticas |
| **QRV** | Está pronto? | Estou pronto |
| **QRT** | Deve parar de transmitir? | Pare de transmitir / Encerro |
| **QSA** | Qual é a intensidade do meu sinal? (1–5) | Intensidade do seu sinal é _____ |
| **QSB** | Meu sinal está desvanecendo? | Seu sinal está desvanecendo |
| **QSY** | Devo mudar de frequência? | Mude para _____ kHz/MHz |
| **QRZ** | Quem me chama? | Você é chamado por _____ |
| **QSO** | Pode comunicar-se com _____? | Posso comunicar-me com _____ |

**Nota sobre "CL" no final do QSO:**
Ao encerrar definitivamente uma sessão de operação (desligar a estação), usa-se **CL** (Closing) após a última identificação. Durante a operação, quando apenas encerrando um contato mas continuando a ouvir, usa-se **SK** (End of contact).

## 6.4 QSL (Cartão de Confirmação de Contato)

O **QSL** é a confirmação de um contato de rádio (QSO). Pode ser feita de três formas:

1. **Cartão QSL físico**: um cartão postal com os dados do contato (data, hora UTC, frequência, modo, sinal recebido) enviado pelo correio. É uma tradição histórica do radioamadorismo e muito apreciada pelos colecionadores.

2. **QSL eletrônico (eQSL)**: serviço online (eQSL.cc) que permite a troca de confirmações digitais.

3. **LOTW (Logbook of the World)**: sistema da ARRL que valida confirmações digitalmente com assinatura criptográfica. É o padrão aceito para premiações como o DXCC.

Os cartões QSL são importantes para diplomas e premiações de radioamadorismo, que geralmente exigem comprovação dos contatos realizados.

## 6.5 Conteste e QRP

### Conteste (Contest)
Competições de radioamadorismo onde o objetivo é fazer o maior número de contatos em um período determinado, seguindo regras específicas (faixas, modos, intervalos, multiplicadores, etc.).

Principais contests internacionais: CQ WW DX, ARRL DX, IOTA Contest, CQWPX.
No Brasil: LABRE-SP Contest, CW Open.

### QRP (Baixa Potência)
Operação com potência muito reduzida. A definição internacional de QRP é:
- Fonia: até 10W de potência de saída
- CW: até 5W de potência de saída

A operação QRP é um desafio técnico e operacional que muitos radioamadores apreciam. Há atividades, competições e diplomas específicos para operação QRP.

:::tip
A operação QRP demonstra excelência técnica: quando você consegue fechar um contato intercontinental com 5W, significa que sua antena, linha de transmissão e técnica operacional são excepcionais.
:::
