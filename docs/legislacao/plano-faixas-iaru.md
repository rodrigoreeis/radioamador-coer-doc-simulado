# Plano de Faixas para a Região 2 (IARU)

## O que é a IARU

A **IARU** (International Amateur Radio Union – União Internacional de Radioamadores) é a organização que representa as sociedades de radioamadores junto à UIT e a outras entidades internacionais. A IARU:

- Tem mais de 160 sociedades-membro em todo o mundo
- Foi fundada em **Paris, em 1925**
- Representa os radioamadores nas conferências internacionais de radiocomunicações
- Publica os planos de faixas voluntários para cada região do mundo

No Brasil, a entidade membro da IARU é a **LABRE** (Liga de Amadores Brasileiros de Rádio Emissão).

## As Três Regiões da IARU

| Região | Países |
|--------|--------|
| **Região 1** | Europa, África, Oriente Médio e Ásia Setentrional (Rússia europeia e asiática) |
| **Região 2** | Américas (América do Norte, Central e do Sul) |
| **Região 3** | Ásia-Pacífico (Ásia Oriental, Sudeste Asiático, Oceania) |

O Brasil está na **Região 2** e, portanto, segue o Plano de Faixas da IARU para a Região 2.

## Indicativos de Chamada por País (Seleção – Região 2)

O prefixo do indicativo de chamada identifica o país de origem da estação:

| País | Prefixos atribuídos |
|------|--------------------:|
| **Argentina** | LO–LW, AY–AZ |
| **Brasil** | PPA–PYZ, ZVA–ZZZ |
| **Canadá** | CF–CK, VA–VG, VO, VX–VY, XJ–XO |
| **Chile** | CA–CE, XQ–XR |
| **Estados Unidos** | AA–AL, K, N, W |
| **Uruguai** | CV–CX |

## Plano de Faixas Voluntário para a Região 2

O Plano de Faixas da IARU é **voluntário** – não tem força de lei, mas é amplamente seguido pela comunidade de radioamadores como questão de ética e organização operacional. Ele divide cada faixa em segmentos com usos típicos recomendados.

**Exemplos de segmentação (simplificado):**

### Faixa de 40 metros (7,000–7,200 MHz)
| Segmento | Uso típico recomendado |
|----------|----------------------|
| 7,000–7,025 kHz | CW exclusivo |
| 7,025–7,040 kHz | CW e modos digitais |
| 7,040–7,050 kHz | Modos digitais prioritários |
| 7,050–7,100 kHz | Fonia (SSB) |
| 7,100–7,200 kHz | Fonia (SSB), centros de atividade |

### Faixa de 20 metros (14,000–14,350 MHz)
| Segmento | Uso típico recomendado |
|----------|----------------------|
| 14,000–14,025 kHz | CW exclusivo |
| 14,025–14,070 kHz | CW e modos digitais |
| 14,070–14,099 kHz | Modos digitais |
| 14,100 kHz | NCDXF/IARU beacons |
| 14,101–14,149 kHz | Modos digitais e automáticos |
| 14,150–14,350 kHz | Fonia (USB) |
| 14,225 kHz | SSTV (centro de atividade) |
| 14,300 kHz | Maritimo móvel, emergências |

### Faixa de 2 metros (144–148 MHz)
| Frequência/Segmento | Uso típico recomendado |
|--------------------|----------------------|
| 144,000–144,025 MHz | EME (Earth-Moon-Earth) CW |
| 144,025–144,150 MHz | EME, SSB fraco sinal |
| 144,150–144,400 MHz | SSB e CW geral |
| 144,400–144,500 MHz | Balizas |
| 144,500–145,800 MHz | Misto (FM simplex, digital, APRS) |
| 145,800–146,000 MHz | Satélites |
| **146,520 MHz** | **Frequência nacional de chamada FM** |
| 147,000–148,000 MHz | Repetidoras e operação FM |
