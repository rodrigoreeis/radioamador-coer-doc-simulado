# Seção 9 – Guia Rápido: Checklist Final

## 9.1 Checklist prático (para o iniciante)

Use este checklist para garantir que você está no caminho certo:

### A. Habilitação do Operador (COER)
- [ ] Estudei o conteúdo das matérias exigidas para a classe desejada
- [ ] Criei minha conta gov.br com verificação de identidade
- [ ] Acessei o SEC (Sistema de Exames de Certificação)
- [ ] Me inscrevi no exame para a classe desejada
- [ ] Preparei meu ambiente para a prova online (câmera, microfone, internet estável)
- [ ] Realizei a prova e fui aprovado
- [ ] Solicitei a emissão do COER via SEI
- [ ] Recebi o COER com meu indicativo de chamada

### B. Licenciamento da Estação
- [ ] Possuo COER válido
- [ ] Abri processo no SEI para licenciamento de estação
- [ ] Informei o tipo de estação (fixa, móvel/portátil, repetidora)
- [ ] Paguei a taxa aplicável (PPDUR/TFI), se necessária
- [ ] Recebi a licença de estação com o indicativo confirmado
- [ ] A licença está com data de validade dentro do prazo

### C. Equipamentos
- [ ] Meu transceptor é homologado pela ANATEL (ou é fabricação caseira com homologação simplificada)
- [ ] A antena está instalada de forma segura
- [ ] Tenho cabo coaxial adequado para a faixa de frequência utilizada
- [ ] Medi a ROE/SWR da antena (valor ideal próximo de 1:1)
- [ ] Sei a potência máxima permitida para minha classe

### D. Operação Diária
- [ ] Sei em quais faixas e segmentos posso transmitir com minha classe
- [ ] Conheço os limites de potência para cada faixa
- [ ] Sei identificar minha estação corretamente (indicativo completo)
- [ ] Identifico minha estação pelo menos a cada hora de operação
- [ ] Ouço a frequência antes de transmitir
- [ ] Conheço as frequências de chamada e centros de atividade das faixas que uso

### E. Emergências
- [ ] Sei reconhecer uma chamada de socorro no rádio
- [ ] Sei como reagir a um "BREAK BREAK" (emergência)
- [ ] Conheço a frequência prioritária de emergência das faixas que uso

### F. Situação Regulatória
- [ ] Verifico periodicamente minha situação no SCRA
- [ ] Mantenho os dados da minha licença atualizados
- [ ] Sei quando minha licença vence e planejo a renovação com antecedência
